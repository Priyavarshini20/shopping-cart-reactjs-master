# Shopping Cart

Built this Ecommerce Website using modern web technologies, including React.js for building the frontend, Tailwind CSS for styling, and the Fakestore API as the data source. 

## Features
- Product Listing
- Filter Functionality
- Shopping Cart
- Checkout
- Responsive Design

## Tech Stack
- React.js
- Tailwind CSS
- Fakestore API
- React Router
